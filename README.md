# Slimenify

## Overview

Slimenify is a mental wellness desktop application designed to help
users better understand and manage their psychological well-being. The
platform combines interactive psychological quizzes, therapist services,
community interaction, and AI-powered features to provide a
comprehensive mental health support environment.

The application allows users to evaluate their emotional state through
structured assessments while connecting them with therapists, events,
and a community forum. Slimenify also provides management tools for
therapists, appointments, quizzes, and users, making it a complete
mental health support system.

## Features

### Mental Health Quizzes

-   Create and manage psychological quizzes
-   Structured question management
-   AI-assisted question suggestions
-   Quiz result tracking and evaluation

### Therapist Management

-   Manage therapist profiles
-   View therapist information and availability

### Appointment Scheduling

-   Book appointments with therapists
-   Manage appointment history and scheduling

### Events

-   Create and manage mental health awareness events
-   Allow users to explore and participate in events

### Community Forum

-   Community space for discussions about mental health
-   Users can share experiences and interact with others

### User Management

-   User registration and authentication
-   User profile and account management

### AI Integration

-   AI-powered suggestions for quiz questions
-   Intelligent support for psychological assessments

## Architecture

Slimenify follows a modular architecture that separates the application
into multiple layers:

-   **Presentation Layer** -- JavaFX UI components and FXML views
-   **Controller Layer** -- Handles user interactions and application
    logic
-   **Service Layer** -- Business logic and application services
-   **Data Access Layer** -- Database communication using JDBC
-   **Database Layer** -- MySQL database for persistent storage

This architecture ensures maintainability, scalability, and clear
separation of concerns.
